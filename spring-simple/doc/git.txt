github 사용법

1. github.com/자신의아이디 로 접속한다.

2. 로그인한다.

3. Repositories 메뉴를 선택한다.

4. New 버튼을 클릭해서 저장소를 만든다.

5. 저장소이름을 만들고 
	맨아래 Initialize this repository with a README 를 체크한다.

	참고]
		public과 private 의 차이점은
		소스코드의 공개여부를 결정한다.
		public 은 접근하는 모든 사람들에게 공개하고
		private은 공동작업자들에게만 공개한다. 요금을 지불할 수도 있다.

6. create Repository 버튼 클릭한다.

7. Branch:master 버튼을 클릭한다.

8. 입력란에 만들브랜치 이름으로 branch 라고 적으면
	아래쪽에 버튼이 생기는데 클릭한다.
   ==> 브랜치가 만들어졌다.

9. 위쪽메뉴에서 branches 메뉴를 선택한다.

10. Default branch 를 확인하면 master 되어 있는데 
    오른쪽에 Change default branch 버튼을 클릭한다.

11. 기본 브랜치 변경하는 화면으로 이동이 되면
    Default branch 섹션에서 
    master 버튼을 클릭하고
    우리가 만든 branch 를 선택하고
    Update 버튼을 클릭한다.
    경고메시지가 띄워지는데 확인버튼을 클릭한다.

12. Collaborators 라는 왼쪽 메뉴를 선택하고
    같이 작업할 공동 작업자들의 아이디를 입력해준다.
    공동작업자는 추가하면 메일이 발송이 되는데
    발송된 메일을 확인해줘야 
    정상적으로 공동으로 작업을 할 수 있다.
